minikernel

View on GitHub

minikernel is a very small kernel for the Raspberry Pi 4 (Rpi4 in the following), written in Rust. The project is still under development, and currently it can use the Rpi4's GPIO pins and transmit/receive messages over miniUART. I'm looking to add support for running applications atop of the kernel, and creating some kind of driver to write to the screen.
