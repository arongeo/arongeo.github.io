shorturl by mrmalac

Check it out on GitHub

I written shorturl in one night. It isn't the best

URL shortener, since it doesn't store URLs in a

Database, but instead, each in its own HTML file.

I know, that it isn't the best solution, but it works, and

when I written it, I didn't know how to work with Databases.

Also, even though it isn't the best performance- and storagewise,

I still find the solution pretty clever.
